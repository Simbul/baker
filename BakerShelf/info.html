About Baker

This is some really useful information for users of your publication.

You can put info here about your publication, subscription model, or how awesome you are.

How do I change this text?

The file info.html in the BakerShelf folder is a simple HTML file that displays right here. Edit it to your heart's content.

If you want to remove it entirely, simply disable the Info button in the - (void)viewWillAppear:(BOOL)animated method.
